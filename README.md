# Pond Mobile Payment Gateway

Secure payment integration using Authorize.net Hosted Payment Page.

## Tech Stack

- **Backend**: Flask (Python 3.12+)
- **Webserver**: Gunicorn 
- **Payment**: Authorize.net API
- **Frontend**: Vanilla HTML/JS
